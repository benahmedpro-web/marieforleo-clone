## Interaction Model
- **Navigation**: Click-driven (menu opens on hamburger click)
- **Hero**: Static with scroll indicator
- **Press logos**: Static grid
- **Forms**: Standard form interaction
- **Podcast carousel**: Scroll or click-to-navigate carousel
- **Success stories**: Click carousel controls
- **Overall page**: Vertical scroll, potentially some scroll-triggered animations on specific elements

## Responsive Breakpoints
- Mobile: 390px
- Tablet: 768px
- Desktop: 1440px

Key layout changes:
- Mobile: Single column, full-width sections
- Tablet: Begin to show 2-column layouts where appropriate
- Desktop: Full grid layouts, side-by-side sections
